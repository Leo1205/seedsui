  <label>Write</label>

导航

用于头部导航、页签导航。

头部导航

<div class="titlebar">
    <a class="titlebar-button" href="javascript:history.go(-1)">
        <i class="icon icon-arrowleft"></i>
    </a>
    <h1>左右图标titlebar</h1>
    <a class="titlebar-button">
        <i class="icon icon-rdoadd"></i>
    </a>
    <a class="titlebar-button">
        <i class="icon icon-search"></i>
    </a>
</div>

头部导航titlebar以自适应方式布局，在此容器内的元素都将 排成一行 ，而文字区域为自适应的区域， 其它区域越大，文字区域就越小 ，具体示例： titlebar.html 页面。

元素 属性 值 描述
I.导航盒子div class .titlebar 导航盒子
.reverse 盒子的颜色反转
II.文字区域h1 class 无 自动被左侧元素挤压
.text-center 文字水平居中
II.左侧元素a或button class 无 绝对定位于左侧，不占用区域
.left 占用左侧区域
II.右侧元素a或button class 无 占用右侧区域
.right 绝对定位于右侧，不占用区域

页签导航

<ul class="slidebar">
    <li class="tab active">
        <i class="icon icon-build-fill"></i>
        <label class="tab-label">机构</label>
    </li>
    <li class="tab">
        <i class="icon icon-person"></i>
        <label class="tab-label">组织</label>
    </li>
    <li class="tab">
        <i class="icon icon-fav"></i>
        <label class="tab-label">收藏</label>
    </li>
</ul>

页签导航slidebar|rectbar|arrowbar|menubar使用table-cell布局，默认为 自适应 ，也可以通过data-col每个页签 强制宽度 ，具体示例： tabbar.html

元素 属性 值 描述
I.页签盒子ul class slidebar 滑片型
rectbar 区块型
arrowbar 箭头型
menubar 菜单型
data-col [2-6] 定义一行的显示的列数，并且强制等宽
II.页签li class tab 页签
active 激活状态

按钮

SeedsUI为系统按钮默认加了一套样式。根据不同场景使用不同的按钮，示例见button.html

按钮

<button class="button radius4 submit">
    <i class="icon icon-rdohook"></i>
    <label>提交</label>
</button>
元素 属性 值 描述
I.按钮button,input[type=button],a class button 主样式
outline 边框型
submit 提交按钮
cancel 取消按钮
disable 禁用按钮
info 信息按钮
success 成功按钮
warn 警告按钮
grayscale 无色按钮
white 白色按钮

徽章

提供宽形徽章、圆形徽章、边角徽章等

宽形徽章

class="badge"，当class中没有设定宽度(sizexx)时，则自适应宽度

<span class="badge size32">1</span>

圆形徽章

class="tip"

<span class="tip">1</span>

边角徽章

class="sticker left|top|right|bottom"

<div class="sticker top right"><span>NEW</span></div>

圆点徽章

class="dot"

<div class="dot"></div>

分组

分组是布局上常用的方式，SeedsUI提供了块级分组、卡片分组和一行布局，我们鼓励您尝试与其它元素结合使用

块级分组

<div class="group">元素</div>

卡片分组

<div class="card">元素</div>

行分组

<div class="inputbox">
    <label class="inputbox-left">用户名</label>
    <label class="inputbox-right">用户名</label>
</div>

表格

    <div style="width:700px;">
    <table>
      <tbody>

            <th class="text-center"><input type="checkbox"></th>
            <th>姓名</th>
            <th>状态</th>
            <th>性别</th>
            <th>邮箱</th>
            <th>创建时间</th>
        </tr>
        </tbody>

    </div>
</div>

I.div标签 class table-box 表格盒子
